      Pertanyaan yang Sering Ditanyakan

      Apakah merge PDF ini benar-benar gratis?

      Ya, 100% gratis. Tidak perlu mendaftar akun, tidak ada batas jumlah file, dan tidak ada watermark pada hasil unduhan.

      Berapa batas ukuran file yang bisa diupload?

      Setiap file dapat berukuran hingga 50 MB. Tidak ada batasan jumlah file dalam satu sesi merge.

      Apakah file saya aman dan terjaga privasinya?

      File Anda diproses di server kami dan dihapus otomatis setelah 1 jam. Kami tidak menyimpan, membaca, atau membagikan dokumen Anda kepada pihak manapun.

      Bagaimana cara menggabungkan JPG ke PDF?

      Cukup upload file JPG/PNG bersama file PDF Anda, atur urutannya, lalu klik Mulai Merge. Gambar akan dikonversi otomatis ke halaman PDF.

      Bisakah saya mengatur urutan halaman sebelum digabungkan?

      Ya! Setelah upload, Anda dapat menyeret (drag & drop) setiap file untuk mengubah urutannya sesuai keinginan sebelum proses merge dimulai.

      Apakah bisa digunakan di HP / smartphone?
